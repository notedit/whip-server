# whip-server
WebRTC whip  server 



## Install 

```
cd whip-server
npm i 
```

## Run the demo

```
node index.js
```

Open your Chrome and visit  http://localhost:8080/index.html

Click the publish and you will publish a stream through `WHIP`

Open the `chrome://webrtc-internals/`  you will see the WebRTC stats
